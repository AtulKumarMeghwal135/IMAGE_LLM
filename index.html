How multimodal LLMs work

01 / Vision encoder
Image → Embeddings
Your image is split into small patches. Each patch is converted to a high-dimensional vector (embedding) using a vision transformer (ViT).
02 / Cross-modal fusion
Image + Text → Unified tokens
The image embeddings and your text question are projected into the same vector space and concatenated into a single sequence of tokens.
03 / Language generation
Tokens → Answer
The transformer decoder attends over both image and text tokens together using self-attention, generating your answer word by word.
